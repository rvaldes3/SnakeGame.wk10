# SnakeGame.wk10
Classic Snake Game built with python

# Instructions
1. Run python file
2. Window pop-up should appear
4. To play use the following controls...

# Controls
Arrow keys -->
Up, down, left, right

# Rules
- Eat as much food as you can!!   [ little green ovals ]
- do not crash into your tail
- do not go out of bound

# Note
Window pop-up cannot be resized

# To restart 
1. close pop-up window
2. Run python file again
